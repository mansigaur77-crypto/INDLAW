# ⚖️ INDLAW - Legal Information System

## 📌 Project Description
INDLAW is a web-based platform where users can:
- Search laws using name or section
- Check penalties based on violations and state
- View latest legal news
- Register and login securely

## 🚀 Features
- User Authentication (Login/Register)
- Law Search System
- Penalty Checker (State-wise)
- News Section (Legal Updates)
- Profile Management

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
